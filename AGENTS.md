# AGENTS.md

This repository is a manuscript production workspace.

## Primary Goal

Help the author move from a source PDF to a polished, reviewable, chapter-based manuscript and clean export files for Amazon KDP.

## Working Rules

- Never overwrite the original file in `input/`
- Treat `manuscript/chapters/` as the source of truth after import
- Edit one chapter or one clearly-scoped batch at a time
- Preserve author voice
- Record notable concerns in `reviews/`
- Keep changes easy to review in Git

## Preferred Editing Order

1. Import and cleanup
2. Chapter normalization
3. Line edit
4. Consistency review
5. Export cleanup

## Output Expectations

When making editorial changes:

- describe what changed
- note any unresolved questions
- avoid silent developmental rewrites
