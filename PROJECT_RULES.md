# Project Rules

This file defines the editorial guardrails for this specific book.

## Non-Negotiables

- Preserve the author's intent
- Do not flatten the author's voice into generic AI prose
- Do not invent new plot points unless the author explicitly asks for developmental help
- Flag continuity concerns instead of silently rewriting around them
- Keep edits reviewable and reversible

## Editing Priorities

1. Correct extraction errors from the source PDF
2. Fix grammar, spelling, punctuation, and formatting
3. Improve clarity, flow, and readability
4. Preserve tone, character voice, and pacing
5. Prepare clean source files for print and ebook export

## Review Standard

- Propose edits chapter by chapter
- Summarize major changes after each chapter
- Call out uncertain lines with clear questions for the author
- Prefer diffs over broad summaries

## Continuity Rules

- Keep names, places, timeline details, and key facts consistent across chapters
- Track unresolved continuity issues in `reviews/continuity-log.md`

## AI Use

- Treat all edits as AI-assisted unless the model writes new original prose that was not already present in the manuscript
- The author remains the final decision-maker on every change
