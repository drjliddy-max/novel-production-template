# Publishing role

The reusable **authoring + editing + voice** framework for every book/article/outreach.

- **Layer:** Editing + Voice.
- **Provides:** authoring pipeline (PDF → Markdown → KDP export), reusable AI `prompts/`,
  `editorial-method/` (portable copyedit/line-edit playbook + scripts), and the **canonical voiceprint**
  `style-guide/JOHN_LIDDY_VOICE.md`.
- **Applied by:** all book repos; voice cited by `../EDITORIAL_STYLE.md`.
- **Operation hub:** `../PUBLISHING.md`.
