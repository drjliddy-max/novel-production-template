# KDP Submission Checklist

Use this during the actual Amazon KDP upload session.

## Before Opening KDP

- [ ] Final manuscript approved by the author
- [ ] Final title and subtitle approved
- [ ] Description approved
- [ ] Categories selected
- [ ] Keywords selected
- [ ] Cover file approved
- [ ] Final manuscript export approved
- [ ] ISBN decision finalized
- [ ] Pricing decision finalized
- [ ] AI disclosure answer decided
- [ ] Rights answer decided

## Book Details

- [ ] Language entered correctly
- [ ] Title matches cover exactly
- [ ] Subtitle matches cover exactly
- [ ] Series information correct
- [ ] Edition information correct
- [ ] Author and contributor names correct
- [ ] Description pasted and checked for formatting
- [ ] Publishing rights confirmed accurately
- [ ] Keywords entered
- [ ] Categories entered
- [ ] Age/range fields reviewed if applicable

## Content Upload

- [ ] Correct manuscript file uploaded
- [ ] Correct cover file uploaded
- [ ] Table of contents checked if ebook
- [ ] Trim size checked if print
- [ ] Paper/ink options checked if print
- [ ] ISBN entered or KDP ISBN selected if print
- [ ] Previewer opened
- [ ] Preview warnings/errors resolved
- [ ] Digital proof reviewed

## Rights And Pricing

- [ ] Territories choice correct
- [ ] Primary marketplace correct
- [ ] Price entered correctly
- [ ] Expanded distribution decision confirmed if offered
- [ ] KDP Select decision confirmed if ebook

## Final Review

- [ ] Cover preview looks correct
- [ ] Interior preview looks correct
- [ ] Metadata is consistent
- [ ] AI disclosure answer confirmed
- [ ] Rights answer confirmed
- [ ] Submit clicked only after author approval

## After Submission

- [ ] Confirmation captured
- [ ] Expected review timeline noted
- [ ] Any KDP follow-up issues tracked
- [ ] Repo updated with final submission notes
