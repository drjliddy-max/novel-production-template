# Amazon KDP Notes

Use this file to track publishing constraints while preparing exports.

## General Guidance

- Keep a clean manuscript source separate from build output
- Prepare ebook and print exports separately when needed
- Review final files in Amazon KDP preview tools before submission

## AI Disclosure

- AI-assisted editing is typically distinct from AI-generated content
- If AI creates substantial original text, review KDP disclosure questions carefully at upload time
- The author should answer KDP upload questions conservatively and accurately

## Working Checklist

- [ ] Front matter reviewed
- [ ] Chapter titles and order confirmed
- [ ] Scene breaks render correctly
- [ ] Paragraph formatting reviewed
- [ ] DOCX export opens cleanly
- [ ] Print-ready PDF strategy confirmed if needed
- [ ] Metadata, categories, and book description drafted

## Final Validation

- Confirm trim size and print formatting separately for paperback or hardcover
- Confirm ebook navigation and table of contents separately for Kindle
- Run one final proofreading pass on exported files, not just source Markdown
