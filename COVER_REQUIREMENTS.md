# Cover Requirements

Use this file to prepare the cover workflow before upload.

## Formats

Possible editions:

- Kindle eBook
- Paperback
- Hardcover

Each may require different cover assets or dimensions.

## Core Decisions

- [ ] Ebook only or print also
- [ ] Paperback needed
- [ ] Hardcover needed
- [ ] Trim size chosen
- [ ] Bleed chosen
- [ ] Interior color choice decided
- [ ] Final page count known

## Paperback And Hardcover

For print, cover dimensions depend on:

- trim size
- page count
- paper type
- bleed settings

Use the official KDP cover tools before final design:

- [KDP Cover Calculator](https://kdp.amazon.com/cover-calculator)
- [Create a Paperback Cover](https://kdp.amazon.com/en_US/help/topic/G201953020?hc_location=ufi)

## Spine Rule

Print spine width depends on final page count and paper type.
Do not finalize the print cover until the interior file is stable enough to trust the page count.

## Practical Workflow

1. Finish editorial cleanup
2. Generate near-final interior export
3. Confirm trim and print options
4. Generate official KDP cover template
5. Build cover on top of that template
6. Preview both interior and cover together

## Asset Checklist

- [ ] Front cover art
- [ ] Back cover copy
- [ ] Author name
- [ ] ISBN barcode space handled if needed
- [ ] Final print cover PDF
- [ ] Final ebook cover image
