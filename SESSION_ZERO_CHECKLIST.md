# Session Zero Checklist

Use this during the live onboarding session on the author's Mac.

## Prep

- [ ] Public template repo exists on GitHub
- [ ] Public template repo is marked as a template
- [ ] Author has manuscript PDF available
- [ ] Internet connection is working

## Machine Setup

- [ ] Homebrew installed
- [ ] Git installed
- [ ] Pandoc installed
- [ ] Poppler installed
- [ ] Python installed
- [ ] Node installed
- [ ] VS Code installed
- [ ] Codex installed
- [ ] Claude Code installed if desired

## Accounts And Auth

- [ ] GitHub account created or confirmed
- [ ] GitHub CLI authenticated
- [ ] OpenAI/Codex authenticated
- [ ] Anthropic authenticated if using Claude Code

## Project Setup

- [ ] `~/Desktop/Projects` exists
- [ ] Private book repo created from template
- [ ] Private repo cloned locally
- [ ] Source PDF placed in `input/original-manuscript.pdf`
- [ ] Repo opened in VS Code

## First AI Run

- [ ] First-run prompt pasted into Codex
- [ ] PDF imported into raw text
- [ ] Chapter files created
- [ ] First 1-2 chapters cleaned up
- [ ] HTML export generated
- [ ] DOCX export generated

## End Of Session

- [ ] Initial import committed
- [ ] First editing pass committed
- [ ] Changes pushed to private GitHub repo
- [ ] Author knows which file to open next
- [ ] Author has this docs set bookmarked
